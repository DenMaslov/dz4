### INSTALLATION:
1. git clone https://github.com/DenMaslov/dz.git
2. cd dz
3. pipenv shell
4. pipenv install
5. cd src
6. run main.py
7. Requires module "statistics" and "pytest"

for testing in tests directory run:  pytest

### TESTED WITH:
* Windows 10
* python 3.9
